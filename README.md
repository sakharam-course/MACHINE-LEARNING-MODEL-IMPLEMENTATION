Introduction

This project is developed as part of Task-4 of the CODTECH Python Programming Internship.
The task focuses on implementing a machine learning classification model that can analyze textual data and predict whether a given email is Spam or Ham (Not Spam).
Spam email detection is a real-world application of Natural Language Processing (NLP) and machine learning, widely used in email filtering systems.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Objective

The main objectives of this task are:

- To understand text preprocessing techniques for NLP
- To convert text data into numerical features using vectorization
- To build a machine learning model for spam detection
- To train and test the model using a labeled dataset
- To evaluate the performance of the classification model

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Implementation Details

The implementation is done using Python and Scikit-learn.
Dataset
- Dataset Name: spam_ham_dataset.csv
- Description: Contains labeled email messages classified as spam or ham
- Key Columns:
    - text – Email message content
    - label – Target variable (spam / ham)
Libraries Used
- Pandas – Data loading and preprocessing
- NumPy – Numerical operations
- Scikit-learn – Machine learning model and evaluation
- CountVectorizer / TF-IDF Vectorizer – Text feature extraction

Steps Implemented
1. Load and explore the dataset.
2. Preprocess text data (cleaning and formatting).
3. Convert text into numerical features using vectorization.
4. Split data into training and testing sets.
5. Train a classification model (e.g., Naive Bayes).
6. Predict labels for test data.
7. Evaluate the model using accuracy and classification metrics.

--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Output and Results

- The model successfully classifies emails into Spam and Ham categories.
- Evaluation metrics such as accuracy, precision, recall, and F1-score are used to measure performance.
- The trained model demonstrates effective spam detection on unseen email messages.
- Sample predictions validate the correctness of the model.
